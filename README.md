# Skills CF - Copilot Agent Skills

Colección de agent skills para GitHub Copilot que guían la generación de proyectos siguiendo los estándares internos de la compañía.

## Skills disponibles

| Skill | Descripción |
|-------|-------------|
| `project-naming-standards` | Validación y generación de nombres de repositorios, artefactos, grupos y módulos |
| `angular-project-setup` | Creación de proyectos Angular frontend con generador Yeoman |
| `springboot-project-setup` | Creación y configuración de proyectos backend Spring Boot |

## Estructura

```
{name-skill}/
  references/       # Archivos de referencia adicionales
  SKILL.md          # Archivo principal del skill
```

## Uso

Estos skills son compatibles con GitHub Copilot Chat y GitHub Copilot CLI.
